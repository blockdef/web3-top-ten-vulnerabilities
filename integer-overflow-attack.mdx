## Code Highlighting

Using `rehype-highlight`, you can add syntax highlighting to your code blocks:

```javascript
const greet = (name) => {
  console.log(`Hello, ${name}!`);
};

greet('World');

You can include code blocks in MDX like this:

```javascript
const greet = (name) => {
  console.log(`Hello, ${name}!`);
};

greet("World");
